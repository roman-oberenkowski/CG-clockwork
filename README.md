Mechanical clock visualization for Computer Graphics course

# Requirements:
- Linux
- OpenGL
- GLEW
- GLFW
- Assimp
# Compilation:
`./Makefile`
# Controls
- Mouse to look around
- END to toggle mouse capture
- WASD to move
- LShift/Space to go down/up
- 0-4 to change animation speed
- B to toggle wooden case rendering
